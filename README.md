# Pikabu
Тестовое задание.


## Описание

**Архитектура**
Упрощенная версия MVVM, чтобы по возможности иметь более чистую архитектуру и возможность масштабирования. Упрощение - по минимуму использовал протоколы.

**Менеджер зависимостей**
SwiftPackage, использовал его так как это нативный, простой инструмент не требующий сложно настройки, с помощью него загружаю внешнюю библиотеку Kingfisher. Эта библиотека упрощает процесс загрузки и кеширования изображения.

**Работа с сетью**
Использую нативную библиотеку URLSession. Для проверки наличия интернет соединения использовал URLSessionConfiguration и свойство waitsForConnectivity = true. 
При наличии интернет соединения публикации автоматически загружаются в таблицу. Если нет - отображается надпись про отсутствие интернета. По истечении N времени Алерт предлагает переподключиться.

**Передача управления**
В работе используются паттерны Колбек и NotificationCenter - передаю информацию между вью и контролером, и отслеживаю изменения состояний для дальнейшего оповещения в определенных частях программы.

**UI**
Основной экран публикаций в котором отображаются все и избранные публикации созданы с использованием UITableView, если бы не было условия в поддержке iOS11 можно было бы попробовать реализовать все на UICollectionView, очень здорово Apple их прокачал, либо весь UI сделать используя SwiftUI, есть небольшой опыт работы с этим. Экран выбранной публикации ViewController с UICollectionView для отображения загруженного Изображения. Кнопка сохранить/удалить решил сделать с использованием картинок, на версии до 13 будет браться из Ассетов, для версии выше SFSymbols - с этим инструментом интереснее работать, посчитал что визуально это будет красивее и современее чем просто надписи. 

**Верстка**
Верстку принял решение делать с использование XIB файлов, так как более конкретного задания не стояло. XIB файлы удобны в переиспользовании, что и было сделано для ячейки таблицы.

**Сохранение**
Публикации сохраняются в UserDefaults при нажатии кнопки Сохранить (пустая иконка), сохраненная публикация отображается как закрашенная кнопка.

**Дополнительно**
В работе использовал расширения для UICollectionView и UITableView для более простой их инициализации из XIB файла.
